# 🎸 NOISED Festival — Landing Page Premium

Landing page profissional, **Mobile-First**, moderna e de alta conversão para o **NOISED Festival**, realizado no **Mirante da Vista Soberba em Teresópolis – RJ** (Serra dos Órgãos).

---

## 🛠️ Tecnologias Utilizadas

- **HTML5 Semântico**: Estruturação acessível com meta tags OpenGraph e SEO.
- **CSS3 Moderno**: Variáveis CSS, Glassmorphism, GPU Acceleration (`transform3d`, `will-change`), bordas neon e layout flexível (Mobile First).
- **JavaScript ES6+**: Animações de viewport com `IntersectionObserver`, física de cursor lerp em Desktop, cronômetro regressivo em tempo real e modais de compra de ingresso e FAQ.
- **Google Fonts**: `Bebas Neue`, `Anton`, `Inter` e `Permanent Marker`.

## ⚡ Como Executar Localmente

### Opção 1: Com Python
```bash
python -m http.server 8080
```
Acesse no navegador: `http://localhost:8080`

### Opção 2: Com NPM
```bash
npm start
```
